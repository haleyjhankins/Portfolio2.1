Ballyard Almanac

The Ballyard Almanac, or "BYA" is my final project I completed as a student at The Iron Yard.

As a novice developer and designer, my goal was to create an interface that is simple and easy to use for the average baseball fan. For this project, I used React to create the User Interface, CSS, a CSV for the data, a Google Maps API, and Express to run the server.
